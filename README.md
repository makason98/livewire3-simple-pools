# Livewire 3 Simple Polls

This is a small demo application built with **Laravel 10** and **Livewire 3**.

Users can:

- Create a poll with a custom title  
- Add multiple options dynamically  
- Vote on existing polls  
- See vote counts update instantly without page reload (thanks to Livewire)

Tech stack:

- Laravel 10
- Livewire 3
- Blade templates
- Tailwind CSS (default Laravel frontend)
